2-SAT

ϕ=⋀ici\begin{aligned} \phi=\bigwedge_{i}c_i \end{aligned}ϕ=i⋀​ci​​其中∣ci∣=2|c_i|=2∣ci​∣=2
有多项式时间算法：
假设ci=(x∨y)c_i=(x\vee y)ci​=(x∨y)，等价于(xˉ→y)(\bar{x}\to y)(xˉ→y)，可以通过建立 xix_ixi​ 和 xˉi\bar{x}_ixˉi​ 的有向图，连边为(xˉ→y)(\bar{x}\to y)(xˉ→y)，看有没有 xix_ixi​ 和 xˉi\bar{x}_ixˉi​ 在一个强连通分量里，没有就可满足。

TSP

一个完全图，一个最小的环经过所有的点。
动态规划可以在O(n⋅2n)O(n \cdot2^n)O(n⋅2n)解

MST

一个完全图，一个最小的树包括所有的点。
多项式时间 Kruskal，Prim

Hamiltonian Circuit

是否存在回路访问每个点一次
动态规划指数级

Euler Circuit

是否存在回路访问每个边一次
有多项式时间算法

Balenced min-cut

要求分为的两部分不太小。
如(∣c1∣>n3)∨(∣c2∣>n3)(|c_1|>\frac{n}{3})\vee (|c_2|>\frac{n}{3})(∣c1​∣>3n​)∨(∣c2​∣>3n​)

min-cut

固定一个点作为sss，遍历其他的点设为ttt，算s→ts\to ts→t的最大流。取最小的最大流。

LP

内点法，椭球法

整数规划

例如 ZOE：
Ax=1xi∈{0,1}A x=1\\ x_i\in\{0, 1\}Ax=1xi​∈{0,1}

max indenpendent set

indenpendent set：在图上找一些点使得这些点互相不为邻居
max indenpendent set：indenpendent set 最多能包含多少个点
无多项式时间算法

min vertex cover

是 max indenpendent set 的补集
vertex cover：找一个点集使得任意边存在一个点在点集里
无多项式时间算法

max clique

找一个最大的点集使得点集内的任意两点之间都有边
无多项式时间算法

shortest path

longest simple path

s→ts\to ts→t最长的路（不能重复访问顶点）
无多项式时间算法

Knapsack

背包问题，WWW为背包容量
O(W⋅poly(n))O(W\cdot\text{poly}(n))O(W⋅poly(n))
无多项式时间算法

Subset sum

从集合中找一个子集使得其和正好为WWW
无多项式时间算法
